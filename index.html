showing video and audio encoding via FFmpeg


สวัสดีครับ ผม จิรภัทร เกียรติชัยพิพัฒน์ เลขที่ 10 ห้อง 6/3 เลขประจำตัว 09115 บางทีในเวลาว่างผมจะนำไฟล์ video หรือ audio มา encode ผ่านโปรแกรม ffmpeg

www.gyan.dev/ffmpeg/

ไฟล์สื่อต่างๆ หลายรูปแบบ เช่น .mp4 .mov .mkv .webm ไม่ว่าจะเป็นวีดิโอ หรือไฟล์เสียง ไฟล์เหล่านี้เป็นไฟล์บรรจุ โดยนามสกุลไฟล์เหล่านี้ไม่ได้ควบคุมวิธีการบีบอัดของข้อมูลด้านใน ซึ่งวีดิโอกับเสียงจะเก็บแยกกันเป็น stream รูปแบบการบีบอัดไฟล์สำหรับบรรจุ stream ต่างๆ คือ codec โดยสำหรับวีดิโออาจเป็น H.264 H.265 VP9 AV1 และสำหรับเสียงอาจเป็น FLAC MP3 AAC Vorbis Opus โดยแต่ละ codec มีความสามารถต่างกัน ทั่วไป codec ที่ ใหม่กว่าจะมีประสิทธิภาพสูงกว่า แต่อาจมี support น้อยกว่า นอกจากนี้ยังมีไฟล์ที่เก็บ stream วีดิโอเท่านั้นเช่น .gif หรือ .webp สำหรับ animation
FFmpeg เป็นโปรแกรมที่สามารถเปิดไฟล์เหล่านี้และนำข้อมูลบันทึกลงใน codec อื่นได้ และยังมีความสามารถการ edit เสียงและวีดิโออื่นๆ อีกด้วย ทำให้เป็นโปรแกรมที่เหมาะกับการใช้เปลี่ยน codec ของสื่อต่างๆ ให้ เป็น codec ที่ทันสมัยมากขึ้น และประหยัดพื้นที่ในการเก็บไฟล์

ตัวอย่างการ Encode video รูปแบบ animation ยาว 12 วินาที โดยเทียบ GIF, webp และ AVIF (เรียงจากเก่าไปใหม่) ทั้ง 3 format นี้สามารถใช้บน Discord ได้ - กดปุ่มเพื่อดู

ปกติผมใช้ทำอะไร และ อุปกรณ์อื่นๆ

บ่อยครั้งผมต้องการที่จะดาวน์โหลดวีดิโอต่างๆ จาก YouTube โดยมีคุณภาพสูง หลายๆคนอาจเคยใช้เว็บไซต์อื่นที่สามารถดาวน์โหลดวีดิโอได้ แต่ผมพบแอป command line คือ yt-dlp ซึ่ง ทำงานบนเครื่องของเราเอง ซึ่งทำงานด้วย Python และสามารถเลือกการดาวน์โหลดได้หลายแบบ เช่น ความละเอียดภาพ bitrate เสียง และเลือก codec ได้อีกด้วย คือ H.264 VP9 AV1 AAC ฟละ Opus นอกจากนี้ยัง support การใช้ร่วมกับ FFmpeg ทำให้สามารถ edit วีดิโอหรือเปลี่ยน codec หลังดาวน์โหลดได้เลย


github.com/yt-dlp/yt-dlp


บางครั้งผมก็สนใจการ extract ข้อมูลสื่อต่างๆ จากเกมที่สนใจ เช่น Elden Ring Nightreign โดยเพลงตัวอย่างที่ใชในเว็บไซต์นี้ มาจากเกมนี้ โดยผมได้ทำการ extract ข้อมูลเมื่อหลายเดือนก่อน เนื่องจากเกมเพิ่งออกมา ทำให้ Soundtrack ไม่อยู่บน Spotify หรือ platform อิ่นๆ ผมจึงมีความสนใจในการ extract สื่อเพลง ออกมาจากเกมเอง

หนึ่งในโปรแกรมที่สำคัญในการ Extract ข้อมูลจาก Elden Ring Nightreign คือ Nuxe โดย JKAnderson ซึ่งสามารถ unpack เกมออกเป็นไฟล์แยก ซึ่งยังคงถูก compress ใน format พิเศษอยู่ คือไฟล์ .bnk


github.com/JKAnderson/Nuxe

github.com/ividyon/WitchyBND

หลังจากแตกไฟล์ของเกมแล้ว ได้ไฟล์ที่บีบอัดแบบ .bnk ซึ่งหลังจากนั้นสามารถใช้โปรแกรม WitchyBND โดย ividyon เพื่อแตกไฟล์อีกครั้ง ได้เป็นไฟล์ .wem ซึ่งคือไฟล์เสียงแต่ใช้ codec เฉพาะทางสำหรับเกม ทำให้แอปส่วนใหญ่ไม่สามารถเล่นได้

ขั้นตอนสุดท้ายในการ Extract เพลงเหล่านี้คือการใช้แอป command line อีกตัวคือ vgmstream ซึ่งสามารถแปล codec ที่เกมจำนวนมากใช้เป็น wav แทน ซึ่งเป็น format ที่ใช้บ่อยของไฟล์เสียง ทำให้สามารถเล่นได้ หรือใช้ ffmpeg เปลี่ยนเป็น codec อื่นต่อไปได้


https://vgmstream.org/

หลังผ่านมาหลายเดือน Soundtrack ของเกมก็ปล่อยออกมา ทำให้เพลงส่วนใหญ่ที่ได้ Extract ออกมาไม่จำเป็นต้องใช้ เนื่องจากเพลงสามารถเล่นจาก streaming service ต่างๆ เช่น Spotify ได้แล้ว แต่ยังคงมีเพลงจำนวนน้อยบางเพลงที่ไม่ได้ถูกปล่อยออกมาอย่างเป็นทางการ ทำให้ยังคงมีบางไฟล์ที่ผมได้ Extract ออกมาที่ยังคงหาได้ยากอยู่ ทำให้ผมสามารถใช้ FFmpeg เพื่อรวมไฟล์เสียงเข้ากับรูปนิ่ง และอัปโหลดเข้า YouTube ซึ่งตอนนี้มียอกวิว 17.5k หลังผ่านมา 1 ปี
